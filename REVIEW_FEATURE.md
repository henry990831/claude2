# 🔍 失误分析与复盘功能

## 功能概述

新增的"失误分析"功能可以在游戏失败后，帮助你找出本局游戏中的关键错误，提升扫雷技巧！

---

## 🎮 如何使用

### 1. 游戏失败后
当你踩到地雷导致游戏结束时，结算画面会显示：

```
💥 游戏失败
用时: 1分32秒
总操作: 45 次
发现失误: 3 处

🔍 查看失误分析 (3)
```

### 2. 点击"查看失误分析"按钮
进入复盘界面，查看详细的失误列表。

### 3. 高亮显示
点击每个失误旁边的"📍 高亮显示"按钮，棋盘会自动滚动到对应位置并高亮 3 秒。

---

## 📊 失误类型

### 💥 致命失误（红色）
**你踩到的那颗地雷**
- 显示位置（行列）
- 发生时间
- 可点击高亮显示

**示例**：
```
💥 致命失误
第 1分32秒
踩到地雷导致游戏失败
位置：行7 列12
[📍 高亮显示]
```

---

### ⚠️ 漏标地雷（黄色）
**本应该先标旗的地雷**

当某个数字周围的隐藏格子数量 = 剩余雷数时，这些格子都应该先标旗。

**示例**：
```
⚠️ 漏标地雷
第 1分28秒
数字 3 周围的隐藏格子都应该先标旗
位置：行6 列11
[📍 高亮显示区域]
```

**解释**：
- 假设有个数字"3"
- 周围已经标了 0 个旗
- 还有 3 个隐藏格子
- 说明这 3 个都是雷，应该全标上旗

---

### 🚩 错误标旗（橙色）
**标错了位置**

你标记了非地雷位置，这可能会干扰判断。

**示例**：
```
🚩 错误标旗
共 2 处
标记了非地雷位置，可能干扰判断
行3 列5、行8 列9
```

**说明**：
- 游戏结束后会检查所有旗子
- 标在非地雷位置的旗子会被列出

---

### 💡 漏掉的安全格（绿色）
**本可以安全点击的格子**

当某个数字周围的旗子数 = 数字本身时，周围其他格子都是安全的。

**示例**：
```
💡 漏掉的安全格
第 1分20秒
数字 2 周围已标够旗，其他格子是安全的
位置：行5 列8
[📍 高亮显示区域]
```

**解释**：
- 有个数字"2"，周围已经标了 2 个旗
- 那么周围其他没标旗的格子都是安全的
- 可以直接点开或使用"和弦"（双击数字）

## 🎯 高亮功能

### 单格高亮
点击"📍 高亮显示"后：
- 🔴 红色发光边框（致命失误）
- 🟡 黄色发光边框（漏标地雷）
- 🟢 绿色发光边框（安全格）

- 3 秒后恢复正常

### 区域高亮
点击"📍 高亮显示区域"后：

- 帮助理解上下文

## 🎓 如何利用复盘提升技巧

### 1. 关注"漏标地雷"
这是最常见的失误类型：
- 看到数字后，先数周围的隐藏格子
- 如果隐藏格子数 = 剩余雷数，全部标旗
- **不要猜！要基于逻辑推理**

### 2. 学会使用"和弦"
当数字周围的旗子够了：
- 双击数字（或单击已揭示的数字）
- 自动打开周围所有未标旗的格子
- 大幅提升速度

### 3. 避免错误标旗
标旗前要确认：
- 是否有明确的逻辑推理
- 不要"随便标一个试试"
- 错误标旗会导致后续判断失误

### 4. 减少漏掉的安全格
看到数字时：
- 先检查周围是否已经标够旗
- 如果够了，周围都是安全的
- 可以快速展开

## 💡 使用技巧

### 技巧 1：失败后立即复盘
趁热打铁，印象最深刻。

### 技巧 2：反复查看同一类失误
如果经常犯同一类错误，重点关注这类提示。

### 技巧 3：练习逻辑推理
- 简单模式：练习基础推理
- 中等模式：练习复杂推理
- 困难模式：练习综合应用

### 技巧 4：降低失误率
目标：
- 新手：失误 < 5 次
- 进阶：失误 < 3 次
- 高手：失误 < 1 次
- 大师：0 失误

## 🔧 技术细节

### 记录的数据
每次操作都会记录：
```javascript
{
  type: 'reveal',  // 操作类型：reveal/flag/unflag/chord
  r: 5,            // 行
  c: 8,            // 列
  time: 92,        // 游戏时间（秒）
  timestamp: ...   // 真实时间戳
}
```

### 分析算法
失败后自动分析：
1. 检查致命位置周围的数字
2. 计算每个数字周围的雷数和隐藏格数
3. 对比旗子数量和实际雷数
4. 找出本可避免的失误

### 性能优化
- 只在失败时分析
- 最多显示前 10 个失误
- 高亮动画使用 CSS transform
- 自动清理旧的高亮效果

## ❓ 常见问题

### Q: 胜利后也有复盘吗？
A: 目前只在失败时提供分析。胜利时会显示总操作数，未来可能加入"完美通关分析"。

### Q: 为什么有些失误没检测到？
A: 目前只能检测明显的逻辑错误。需要高级推理的情况（如排除法、假设法）暂时无法分析。

### Q: 可以关闭这个功能吗？
A: 可以。失败后直接点"再玩一次"或"返回首页"，不点"查看失误分析"即可。

### Q: 失误数会影响金币吗？
A: 不会。失误分析只是教学功能，不影响奖励。

### Q: 能看历史局的复盘吗？
A: 目前只能看当前局。历史记录功能正在规划中。
