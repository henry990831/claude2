# 💎 扫雷 Minesweeper Pro - Enhanced Edition

一个现代化、功能丰富的扫雷游戏 WebApp，基于 Google Apps Script 部署。

## ✨ 核心特性

### 🎮 游戏模式

1. **经典模式**
   - 简单（9×9，10雷）- 基础奖励 5 金币
   - 中等（16×16，40雷）- 基础奖励 15 金币
   - 困难（30×16，99雷）- 基础奖励 30 金币
   - 自定义（自由设置行列和地雷数）

2. **🔥 生存模式**（NEW!）
   - 连续通关，难度逐步递增
   - 连胜加成：每次连胜金币奖励 +10%
   - 失败一次，连胜归零，从头开始
   - 连胜里程碑：5/10/20 次解锁特殊成就

3. **🧘 禅模式**（NEW!）
   - 无计时压力，专注享受扫雷乐趣
   - 舒缓音效和平静动画
   - 少量金币奖励（5金币/局）
   - 轻松（9×9）和平和（16×16）两种难度

### 📅 每日签到系统（NEW!）

- **时区支持**：自动识别用户时区
- **自定义重置时间**：默认每天 0 点重置（可配置）
- **连续签到奖励**：
  - 第1天：10 金币
  - 第2天：15 金币
  - 第3天：20 金币
  - 第4天：30 金币
  - 第5天：40 金币
  - 第6天：60 金币
  - 第7天：100 金币（循环）
- **签到记录**：查看历史签到情况
- **中断处理**：断签后从第1天重新开始

## 🎯 金币获取方式

1. **通关奖励**
   - 基础金币（根据难度）
   - 速度加成（标准时间50%内 ×3.0）
   - 道具使用惩罚（每使用一次 -15%）
   - 无道具奖励（+50%）
   - 每日首胜（+20🪙）

2. **生存模式加成**
   - 连胜加成：每次连胜 +10%
   - 例如：5连胜时获得基础金币 ×1.5

3. **每日签到**
   - 连续签到 7 天最高可获 175 金币

4. **成就奖励**
   - 完成成就可获得 50-500 金币

5. **禅模式**
   - 每局固定 5 金币（轻松赚钱方式）

## 🔧 技术实现

### 数据存储
- 使用 `localStorage` 本地存储
- 存档版本：`ms_save_v2`
- 自动向后兼容旧版本存档

### 时区处理
```javascript
// 自动识别用户时区
timezone: Intl.DateTimeFormat().resolvedOptions().timeZone

// 支持自定义重置时间（默认0点）
resetHour: 0  // 可配置为任意小时
```

### 音效系统
- Web Audio API 合成音效
- 无需外部音频文件
- 多种波形（sine, square, triangle, sawtooth）

### 振动反馈
```javascript
// 使用 Vibration API
navigator.vibrate([pattern])
```
