# 💎 扫雷 Minesweeper Pro - 完整功能清单

## ✅ 已实现功能

### 🏆 段位系统
- **6个段位**：`◆ 青铜` → `◇ 白银` → `✦ 黄金` → `❖ 钻石` → `✧ 大师` → `✺ 王者`
- **计算公式**：`段位分 = (胜场 × 5) + (胜率 × 1000)`
- **专属皮肤**：黄金以上解锁专属主题皮肤
  - 🟡 黄金：黄金星辰主题
  - 💎 钻石：璀璨钻石主题
  - 👑 大师：王者荣耀主题
  - 🔥 王者：永恒火焰主题
- **实时升降**：无保护机制，胜率下降立即降段

### 🎲 特殊地雷模式（第4模式）
- **4种特殊地雷**：
  - 💰 **宝藏雷**(5%)：点击获得5-15金币，不爆炸
  - 🔥 **连锁雷**(10%)：爆炸引爆周围8格地雷
  - ❄️ **冰冻雷**(5%)：不爆炸，暂停计时5秒  
  - 🎲 **随机雷**(5%)：30%金币/25%揭示/20%标记/15%时间-10s/10%爆炸
- **平衡性**：连锁范围限定为周围8格，不继续连锁
- **奖励**：金币 +30%

### 📊 统计中心
1. **游戏统计**
   - 总局数、胜场、败场、胜率
   - 分难度统计（简单/中等/困难）
   - 总游戏时长、平均每局时长
   - 完美通关次数（无道具）

2. **金币统计**
   - 累计获得/消耗/当前余额/历史最高
   - 收入来源分布（通关/签到/成就）
   - 支出分类（道具/皮肤/背景/音效/特效）

3. **成就统计**
   - 已解锁数量/总数/完成度
   - 总成就点数/稀有成就数

4. **连胜统计**
   - 当前连胜/历史最高/生存模式最高关卡
   - 连胜分布（1-5/6-10/11-20/21+）

5. **时间统计**
   - 首次游戏日期/最后游戏日期
   - 活跃天数/连续天数
   - 游戏时间分布（按小时段）
   - 最活跃时段

6. **操作统计**
   - 总点击格子数/标记次数/道具使用次数
   - 道具使用分布

## 🚀 使用指南

### 部署到 Google Apps Script
1. 复制 4 个文件内容
2. 在 Apps Script 中创建对应文件
3. 部署为 Web 应用
4. 开始游戏！

### 段位系统
- 首页右上角显示当前段位
- 点击查看详细段位信息
- 黄金以上自动解锁专属皮肤

### 特殊地雷模式
- 首页选择"特殊地雷"模式
- 选择难度开始游戏
- 注意不同颜色的数字提示

### 统计中心  
- 首页点击"统计"按钮
- 5个标签页切换查看不同维度

### 装饰系统
- 商城购买各种装饰
- 在"装饰"界面装备
- 实时预览效果

### 音乐播放器
- 首页点击"音乐"按钮
- 购买喜欢的曲目
- 调整音量和播放模式

### 护眼模式
- 设置中开启护眼模式
- 选择强度（轻度/中度/重度）
- 设置游戏时长提醒

## 💡 技巧

1. **快速升段**：保持高胜率比刷胜场更重要

3. **金币获取**：每日签到 + 成就 + 高胜率通关
4. **护眼游戏**：开启护眼模式 + 30分钟提醒
