# 🌍 无限世界 RPG (Google Apps Script + Three.js)

第一人称 3D 开放世界 RPG，跑在 **Google Apps Script Web App** 里。**程序化生成**的无限世界（种子驱动，类似无人深空的做法）——地形、生物群系、怪物、资源全部由种子实时算出，永不手工绘制。**越往外走越危险、掉落越多越好。**

3D 渲染由浏览器端 **Three.js** 完成，GAS 仅托管网页。存档存 `localStorage`。

## 📁 文件结构（GAS 项目）

| 文件 | 作用 |
|------|------|
| `Code.gs` | `doGet()` 返回 Index 模板；`include()` 内联 CSS/JS |
| `Index.html` | 页面骨架 + 所有 UI 容器 |
| `CSS.html` | 全部样式（HUD、7品级配色、怪物血条、人物面板等） |
| `JS.html` | 游戏引擎（噪声/世界/渲染/控制/怪物/物品/战斗/背包/人物） |

## 🚀 部署到 Google Apps Script

1. [script.google.com](https://script.google.com) → 新建项目
2. 创建 4 个文件（名称完全一致）：`Code.gs`、`Index`(HTML)、`CSS`(HTML)、`JS`(HTML)，把 `rpg/` 下对应内容粘进去
3. 部署 → 新建部署 → 网页应用 → 访问权限「任何人」
4. 打开生成的网址即可游玩

> ⚠️ GAS 运行在跨域 iframe 中，浏览器指针锁定可能被禁用。本游戏已做回退：**未锁定时移动鼠标照样能转视角**，无需拖拽。
>
> ⚠️ 这是**单机 RPG**（程序化生成 + 本地存档），非实时多人 MMO——GAS 无 WebSocket、有执行超时与配额，不适合实时联机。

## ⚙️ 平衡数值（可调）
- 装备掉率随深度：T1 15% → T5 45%；品级分布随深度上移
- 合成概率（阶段二）：灰+灰 → 60% 灰 / 32% 绿 / 8% 蓝，越高品级越难越级
- 强化：+1~+3 必成，越高成功率越低，失败可用保护石兜底
