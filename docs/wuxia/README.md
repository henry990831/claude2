# 水墨江湖 · 单机武侠 RPG

纯网页(HTML5 Canvas，零依赖)的单机武侠动作 RPG，水墨风。灵感来自《墨香》的江湖气质。
合并到 `main` 后由 GitHub Pages 自动上线，访问地址：

**https://henry990831.github.io/claude2/wuxia/**

## 操作
- **WASD / 方向键**：行走（也可按住鼠标右键朝光标移动）
- **鼠标左键 / J**：普攻（靠近敌人会自动出招，方便手机）
- **1 剑气斩 · 2 掌风 · 3 踏雪(闪)**：武学（消耗内力）
- **B / I**：行囊　**C**：属性

## 已实现（第一版）
- 水墨风世界：宣纸地面、竹/松/石/亭景物、河流、安全城镇（城内快速回血回气）
- 战斗：普攻 + 3 技能、暴击、伤害飘字、水墨粒子特效
- 怪物：野狼/山贼/黑熊，AI（游荡→追击→攻击），血条等级，死亡掉落与重生
- 成长：经验/升级（属性成长）、装备掉落（凡品/精良/稀有/绝世四品阶）
- 物品：兵器/护甲/丹药，行囊、装备栏、使用/装备/卖出/丢弃
- 存档：localStorage 自动存档；开始界面可"继续前缘"
- 小地图、气血/内力/经验 HUD、消息提示
