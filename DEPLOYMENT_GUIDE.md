# 🚀 部署指南 - Deployment Guide

## Google Apps Script 部署步骤

### 第一步：创建项目

1. 访问 [Google Apps Script](https://script.google.com/)
2. 点击 **新建项目**
3. 将项目重命名为 "扫雷 Minesweeper Pro"

### 第二步：添加文件

#### 1. Code.gs（主文件）
```
将 Code.gs 的全部内容复制到默认的 Code.gs 文件中
```

#### 2. Index.html
- 点击左侧的 **+** 按钮
- 选择 **HTML**
- 命名为 `Index`
- 将 Index.html 的内容复制进去

#### 3. CSS.html
- 点击左侧的 **+** 按钮
- 选择 **HTML**
- 命名为 `CSS`
- 将 CSS.html 的内容复制进去

#### 4. JS.html
- 点击左侧的 **+** 按钮
- 选择 **HTML**
- 命名为 `JS`
- 将 JS.html 的内容复制进去

### 第三步：部署为 Web 应用

1. 点击右上角 **部署** → **新建部署**
2. 点击 **选择类型** → **网页应用**
3. 填写信息：
   - **说明**：扫雷 Minesweeper Pro v1.0
   - **执行身份**：我
   - **访问权限**：任何人
4. 点击 **部署**
5. 授权应用（首次部署需要）
6. 复制生成的 **网页应用网址**

### 第四步：测试

1. 在浏览器中打开刚才复制的网址
2. 测试所有功能：
   - ✅ 经典模式游戏
   - ✅ 生存模式
   - ✅ 禅模式
   - ✅ 每日签到
   - ✅ 成就系统
   - ✅ 商城购买
   - ✅ 移动端振动反馈

## 📱 添加到主屏幕（PWA 体验）

4. 确认添加

## 🔄 更新部署

当需要更新代码时：

1. 在 Apps Script 编辑器中修改代码
2. 点击 **部署** → **管理部署**
3. 点击现有部署旁的 **✏️ 编辑**
4. 更改 **版本** 为 "新版本"
5. 更新说明（例如：v1.1 - 新增 XXX 功能）
6. 点击 **部署**

⚠️ **注意**：网址不会变化，用户刷新页面即可看到更新

## 🎨 自定义配置

### 修改开发者密码
在 `JS.html` 中找到：
```javascript
const DEV_PASSWORD = 'TWH0831';
```
改为你想要的密码。

### 修改签到重置时间
在 `createDefaultDaily()` 函数中：
```javascript
resetHour: 0  // 改为任意 0-23 的小时
```

### 调整金币价格
在 JS.html 顶部的常量配置中修改 `ITEMS`、`SKINS`、`BACKGROUNDS` 等对象的 `price` 值。

### 添加新成就
在 `ACHIEVEMENTS` 对象中添加新条目：
```javascript
new_achievement: { 
  id:'new_achievement', 
  name:'成就名称', 
  icon:'🎯', 
  desc:'完成条件描述', 
  reward:100, 
  check:(s)=>s.someCondition >= targetValue 
}
```

## 🛠️ 故障排查

### 问题：部署后页面空白
**解决方案**：
- 检查浏览器控制台是否有 JavaScript 错误
- 确认所有文件名称正确（区分大小写）
- 清除浏览器缓存后重试

### 问题：存档丢失
**解决方案**：
- localStorage 是基于域名的，不同域名的部署会有不同存档
- 检查浏览器是否开启了隐私模式（会导致 localStorage 不可用）
- 可以在控制台运行 `localStorage.getItem('ms_save_v2')` 查看存档

### 问题：振动不工作
**解决方案**：
- 振动功能仅在支持的移动设备上工作
- 需要 HTTPS 连接（Google Apps Script 自动提供）
- 某些浏览器可能需要用户手势触发

### 问题：音效不播放
**解决方案**：
- 某些浏览器需要用户交互后才能播放音频
- 检查设备是否静音
- 查看浏览器控制台是否有 AudioContext 相关错误

## 📊 性能优化

### 减少初始加载时间
1. 在 CSS.html 中移除不需要的动画
2. 减少背景特效的复杂度
3. 压缩 emoji 图标使用

### 提升游戏流畅度
1. 调整动画时长（在 CSS 中搜索 `animation:`）
2. 减少同时显示的格子数（限制自定义模式最大值）
3. 优化 `revealCell` 函数的递归深度
